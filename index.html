Welcome to OpenStreetMap

OpenStreetMap is a free map that you can edit. Here's a quick guide to get you started!

1 One Database - Many Applications

If you're here to correct an error you found on a site like Foursquare or Craigslist, you're in the right place - OpenStreetMap is the central database used by hundreds of websites.

You just need to edit OpenStreetMap, and the changes will be visible everywhere else. Copying from place to place can take a few minutes to a few hours, so don't worry if it isn't instant.

2 What's on the Map

OpenStreetMap is a place for any data that's both real and current - it includes millions of buildings, roads, and other details about places.

What it doesn't include is opinionated data, like ratings, or data that isn't current, like historical buildings or hypothetical structures.

3 Get a Free Account

To edit OpenStreetMap, you'll need a free user account - sign up here.

4 Learn to edit

Editing the map isn't too hard: the easiest way to start is with 'iD Editor', which runs in your browser.
