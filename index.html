Welcome to OpenStreetMap

a free map that you can edit

this is a beta website. if you have a suggestion or bug report, let us know on github

Unlike other maps, OpenStreetMap is completely created by people like you, and free for anyone to fix, update, download and use. Here's a quick guide to get you started!

One Map: Many Websites

If you're here to correct an error you found on a site like Foursquare or Craigslist, you're in the right place - OpenStreetMap is the central map database used by hundreds of websites.

You just need to edit OpenStreetMap, and the changes will be visible everywhere else. Copying the map data from place to place can take from a few minutes to a few hours, so don't worry if it isn't instant.

What's on the Map

OpenStreetMap is a place for any data that's both real and current - it includes millions of buildings, roads, and other details about places.

What it doesn't include is opinionated data, like ratings, or data that isn't current, like historical buildings or hypothetical structures.

No Time To Edit? Add a Note!

If you just want something small fixed and don't have the time to sign up and learn how to edit, it's easy to add a note!

Just go to osm.org and click 'Add a note' in the bottom-right corner. This will add a marker to the map, which you can move by dragging the green plus sign. Click save to add a note, and experienced mappers will find and address your issue.

Get a Free Account

To edit OpenStreetMap, you'll need a free user account - sign up here.

With your user account, you can edit the map, create a profile page, and communicate with other people working on improving the map.

Basic Terms For Mapping

OpenStreetMap has some of its own lingo. Here are a few key words that'll come in handy.

An Editor is a program or website you can use to edit the map.

A Node is a point on the map, like a single restauraunt or a tree.

A Way is a line, like a road, stream, or shape.

A Changeset is group of changes to the map saved by a user.

Any questions?

Need help mapping, or not clear on how to use OpenStreetMap? Feel free to ask on OpenStreetMap's Help website!

Start Editing

Once you have an account, you can click the Edit button on osm.org to edit any part of the world.

Since OpenStreetMap is an open project, there are lots of different ways to edit the map - whether with a web based editor like iD, or on your iPhone with Pushpin, or with your Android phone with OsmAnd+.
